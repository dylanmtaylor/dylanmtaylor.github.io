Introducing FiOS WEP Calculator Mobile!

21 Nov 2010

For all of you who don't have an Android</a> device, it's been difficult to calculate a default FiOS</a> WEP</a> key using your mobile phone</a>. Well not any more! Now if you visit the FiOS WEP Calculator website</a> on any mobile device</a>, as long as your device supports JavaScript</a>, you can calculate a default WEP key (I'm also planning on porting the script to PHP</a>, and using server-side calculations for devices that don't support JavaScript in the future). The mobile optimized version is incredibly</em> lightweight, with no CSS</a> files (all styles are embedded in the main page) or images, what-so-ever, and all unnecessary formatting in the source code removed. On a 3G connection, the page should easily load in under 1 second almost every time (the entire page is less than 3kb). I believe that this new website will make it significantly easier for users of every mobile platform to use FiOS WEP calculator on the go.
